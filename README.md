# sudoku-solver

A Python Sudoku solver I made for fun.

## Running

```python sudoku.py <input_file>```

## Input Files

As the test cases show, the input files are formatted like this:
```
003020600
900305001
001806400
008102900
700000008
006708200
002609500
800203009
005010300
```
Where 0 represents a blank space, and 1-9 represent the starting numbers.
